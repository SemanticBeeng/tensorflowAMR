# tensorflowAMR
Two applications of character-level seq2seq RNN translation.

# Contents
```SemEval2016``` - AMR Parsing with character-level seq2seq RNN. <br/>
```LREC2016``` - character-level English-Latvian seq2seq RNN translation.
